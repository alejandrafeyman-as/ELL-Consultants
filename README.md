# New Company Website

This is a simple Next.js website package prepared for Vercel.

## How to use

1. Create a free GitHub repository.
2. Upload all files from this folder.
3. In Vercel, click **Add New Project**.
4. Import the GitHub repository.
5. Click **Deploy**.

## Customize

Edit these files:
- `app/page.tsx` for the page content
- `app/layout.tsx` for the browser title and description
- `app/globals.css` for basic global styling

## Run locally

```bash
npm install
npm run dev
```
